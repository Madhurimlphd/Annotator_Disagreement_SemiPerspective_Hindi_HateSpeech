This GitHub repository contains the following:


A]	The IDs of revised labels from both iterations, provided in the following files:
* relabel_iter1_text_ids: IDs corresponding to the first iteration
* relabel_iter2_text_ids: IDs corresponding to the second iteration
* These IDs are used to construct the HH_Semi_Perspective resource.

B] The HH-LEX lexicon used in this study is available on Zenodo.
* Link: https://zenodo.org/records/17938022
* Detailed information about the lexicon can be found in the file Lexicon_Information.

C] The detailed workflow of the code utilised in the FI-RC framework is documented in the file Code_Structure.

D] The link to the paper will be shared as soon as it becomes available.
